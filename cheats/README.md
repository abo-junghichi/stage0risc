# cheat tools
stage0プロジェクトの主要目的として
"Reflections on Trusting Trust" からの防御が挙げられるので、
検証不能な大きさのバイナリで供給される
既存ツールへの依存は避けねばならない。
しかし、それはあまりにも面倒くさいので、
このフォルダでは既存ツールに手を出す不正行為を犯す。

## stage0risc用アドレス計算付き16進バイト変換器 hexrel.out
16進バイト変換器hex.outにアドレス計算機能を付け足したもの。
C言語プリプロセッサなどの既存のマクロプロセッサと組み合わせて使えば、
マクロ付きアセンブラ並みの記述性が得られる。
<pre>
$ make hexrel.out
</pre>
で構築できる。
